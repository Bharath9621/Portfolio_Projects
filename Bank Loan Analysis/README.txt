# Bank Loan Analytics & Risk Monitoring Dashboard

## Project Overview

This project focuses on analyzing a bank's lending portfolio to monitor loan performance, assess portfolio quality, identify repayment trends, and evaluate borrower risk. The solution combines SQL, Python, and Power BI to transform raw loan data into actionable business insights.

---

## Business Problem

Banks require continuous monitoring of lending activities to ensure portfolio health and minimize credit risk.

The objective of this project was to:

* Monitor loan applications and disbursements
* Track repayment performance
* Analyze borrower financial health
* Identify good and bad loans
* Support data-driven lending decisions

## Project Workflow

### 1. Data Collection

* Imported loan portfolio dataset
* Reviewed schema and business requirements

### 2. Data Cleaning & Preparation

* Missing value analysis
* Data type validation
* Date transformation
* Feature engineering

### 3. SQL Analysis

Developed SQL queries to calculate:

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average DTI
* Good Loan Metrics
* Bad Loan Metrics
* Loan Status Analysis

### 4. Exploratory Data Analysis (EDA)

Performed Python-based analysis to identify:

* Monthly lending trends
* State-wise funding distribution
* Loan term analysis
* Employment length trends
* Loan purpose segmentation

### 5. Dashboard Development

Built interactive Power BI dashboards to monitor:

* Lending KPIs
* Portfolio quality
* Regional performance
* Borrower behavior

---

## Key KPIs

| KPI                     | Description                   |
| ----------------------- | ----------------------------- |
| Total Loan Applications | Total number of loans issued  |
| Total Funded Amount     | Total loan amount disbursed   |
| Total Amount Received   | Total repayments received     |
| Average Interest Rate   | Portfolio lending rate        |
| Average DTI             | Borrower debt-to-income ratio |
| Good Loan %             | Fully Paid + Current Loans    |
| Bad Loan %              | Charged Off Loans             |

## Key Insights

* Loan applications increased steadily throughout the year.
* California contributed the highest funded amount.
* Debt Consolidation was the most common loan purpose.
* 36-month loans represented the majority of issued loans.
* Good Loans significantly exceeded Bad Loans, indicating a healthy portfolio.
